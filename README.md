# An Chat Bot Demo

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

   Install CocoaPods in `ios` folder

   ```bash
   pod install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

   Or run in iOS simulator

   ```bash
   npm run ios
   ```

## Setup ChatGPT

Add the `OPENAI_URL` and `OPENAI_TOKEN` in `.env` file. For example:

```bash
OPENAI_URL = https://openai_url
OPENAI_TOKEN = xxxxxxxxxxxxxxxxx
```
